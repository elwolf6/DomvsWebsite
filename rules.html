Rules

All actions may be punished by the administration team if considered harmful or gamebreaking. Even if they aren't in the server rules.

Please read before playing.

Rule 1.1

Do not hack or exploit in any way, or attempt to bypass a server rule.

Rule 1.2

Cheat clients, macro keys and other modifications which give the player gameplay leverage over other players are not allowed.

Rule 2.1

Do not intentionally make the server lag.

Rule 2.2

Mob farms, Redstone farms which do not intentionally make the server lag can still be removed.

Rule 3.1

Do not ruin the landscape of the map (Lava casting mountain). This includes griefing unclaimed builds.

Rule 3.1

Do not ruin the landscape of the map (Lava casting mountain). This includes griefing unclaimed builds.

Rule 3.2

Looting of unclaimed builds is allowed.

Rule 3.3

Griefing builds gained through warfare is not allowed.

Rule 4.1

Do not combat log or war log.

Rule 4.2

War may only be declared if the town has a minimum of 2 people online.

Rule 4.3

If the town has less than 2 people in it then message an admin for the best course of action.

Rule 5.1

Do not spam the chat.

Rule 5.2

This includes discord channels and discord relay.

Rule 6.1

Do not use alts for any purpose.

Rule 7.1

Do not use macros that are possibly disruptive to server activity and server performance (things that cause chat spam, etc.)

Rule 8.1

Towns and Nations are not allowed in any other Realm except for the Overworld.

Rule 8.2

Outposts are on the other hand allowed in any other Realm.
